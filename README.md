# Data-Science-Project-Netflix-Data-Analysis

This project is a beginner-friendly Data Science project created for learning purposes using a Netflix movies dataset.

Project Objective
The goal of this project is to analyze Netflix movie data and discover useful insights through:
Data Cleaning
Exploratory Data Analysis (EDA)
Data Visualization
Trend Analysis

Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

Key Analysis Performed
 Most frequent movie genres
 Highest voted movies
 Most popular movies
 Lowest popularity movies
 Year-wise movie release analysis

Conclusion
